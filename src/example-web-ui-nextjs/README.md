# Example Web UI NextJS

An example Next.js web application with an amber and deep purple theme.

## Getting Started

From the repository root, navigate to this folder:

```bash
cd src/example-web-ui-nextjs
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Running Tests

Run the test suite:

```bash
npm test
```
